# KK Book Diary

A minimal, monochromatic web application for creating and sharing book entries (profiles) with photos.

## Features

- Create profiles with name, address, contact, date, note, and photo
- Edit existing profiles
- Delete profiles
- Copy shareable links
- View profiles in a clean, dedicated view
- Download profile photos
- Mobile-friendly responsive design
- Monochromatic minimalist UI with SVG icons

## Tech Stack

- **Backend**: Node.js + Express
- **File Upload**: Multer
- **Storage**: JSON files + file system (no database)
- **Frontend**: Pure HTML/CSS/JavaScript (no frameworks)
- **Icons**: Inline SVG
